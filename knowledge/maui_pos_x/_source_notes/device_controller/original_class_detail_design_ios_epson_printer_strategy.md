## 概要

Epson ePOS SDK（iOS Binding Library）を使用してiOS環境でプリンターを制御するストラテジクラス。
Bluetooth接続およびTCP/LAN接続の両方に対応し、デバイスディスカバリーによる自動検出機能を持つ。
レシート印刷（テキスト・バーコード・QRコード・画像・ロゴ）およびドロワー開放操作を提供する。

##### 戻り値

| 型   | 概要 |
| ---- | ---- |
| void | -    |

##### 引数

| 修飾子 | 型              | 引数名（論理） | 引数名（物理） | 概要                               |
| ------ | --------------- | -------------- | -------------- | ---------------------------------- |
| -      | Epos2DeviceInfo | デバイス情報   | deviceInfo     | Epson SDK から検出されたデバイス情報 |

##### 発生例外

| 型  | 概要 |
| --- | ---- |
| -   | -    |

##### 処理概要

| 項番 | 処理概要                                                                                                                                                   |
| ---- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
