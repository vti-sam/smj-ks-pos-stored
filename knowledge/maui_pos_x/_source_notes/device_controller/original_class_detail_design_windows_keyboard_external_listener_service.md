## 概要

Windows環境におけるPOS専用キーボードからの「生入力（RawInput）」を直接傍受し、アプリケーションに通知するためのクラスです。\
以前の実装では全画面の基底クラスである `BaseContentPage` から直接呼び出されていましたが、**現在は `WindowsRawKeyboardStrategy` (Strategy Pattern) を経由して完全にカプセル化**されています。これにより、OSレベルからの着実なキャプチャと安全なメモリ管理（リソース解放）を両立しています。
（流れ：OSイベント受信 → 内部変換 → Strategyへ通知 → App.xaml.csにて文字列パース → ViewModel呼び出し）
