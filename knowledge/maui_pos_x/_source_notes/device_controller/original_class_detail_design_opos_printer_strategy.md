## 概要

Windows環境でNamedPipe経由でHost側のOPOS OCXを制御し、レシートプリンターを操作するストラテジクラス。
プリンターのライフサイクル管理（接続・切断）、レシート印刷（JSONシリアライズ経由）、およびドロワー開放操作を提供する。

### メソッド名: レシート印刷

#### 戻り値

| 型                       | 概要                         |
| ------------------------ | ---------------------------- |
| Task\<PrinterResponse\>  | 印刷結果を含むプリンター応答 |

#### 引数

| 修飾子 | 型       | 引数名（論理） | 引数名（物理） | 概要                 |
| ------ | -------- | -------------- | -------------- | -------------------- |
| -      | Receipt? | レシートデータ | receipt        | 印刷するレシート情報 |

#### 発生例外

| 型        | 概要                                                       |
| --------- | ---------------------------------------------------------- |
| Exception | JSONシリアライズまたはパイプ通信で例外が発生した場合       |

#### 処理概要

| 項番 | 処理概要                                                                                                  |
| ---- | --------------------------------------------------------------------------------------------------------- |
| 1    | レシートデータ をJSON文字列にシリアライズする。                                                                          |
| 2    | パイプコマンド送信で `OPOS_PrinterPrintReceipt;{JSON}` コマンドを送信する。                                              |
| 3    | 応答が `ERROR_WIN32_` のプレフィックスで始まる場合、`CreateConnectionFailedResponse` で接続失敗レスポンスを返却する。    |
| 4    | 応答を `JsonSerializer.Deserialize<PrinterResponse>` でデシリアライズして返却する。                                      |
| 5    | `JsonException` 発生時やその他デシリアライズ失敗時は、`CreateConnectionFailedResponse` で接続失敗レスポンスを返却する。  |
| 6    | その他例外発生時は `Task.FromException` で伝播する。                                                                     |
