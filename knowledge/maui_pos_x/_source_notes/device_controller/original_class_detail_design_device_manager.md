## 概要

デバイスストラテジーのライフサイクル管理・設定読み込み・デバイス取得を一元管理するシングルトンクラス。
config.jsonからデバイス設定を読み込み、実行プラットフォーム（Windows/iOS/Android）に応じたストラテジーの登録・生成・キャッシュ管理を行う。
StrategyFactoryと連携してデバイス種別ごとのストラテジーインスタンスを提供する。

> ※ private コンストラクタ。シングルトンパターンのため外部からのインスタンス生成不可。`Instance` プロパティからアクセスする。

### メソッド名: アクティブデバイス取得

#### 戻り値

| 型         | 概要                                               |
| ---------- | -------------------------------------------------- |
| DeviceSpec | アクティブデバイス仕様（該当なしの場合は空の仕様） |

#### 引数

| 修飾子 | 型                  | 引数名（論理）       | 引数名（物理） | 概要                       |
| ------ | ------------------- | -------------------- | -------------- | -------------------------- |
| -      | List\<DeviceSpec\>  | デバイス一覧         | devices        | 全デバイス仕様リスト       |
| -      | ActiveDevice        | アクティブデバイス   | activeDevices  | アクティブデバイス設定     |
| -      | string              | デバイス種別         | type           | 取得対象のデバイス種別     |
| -      | string              | OS種別               | os             | 実行OS（ios/android/windows）|

#### 発生例外

| 型  | 概要 |
| --- | ---- |
| -   | -    |

#### 処理概要

| 項番 | 処理概要                                                                                              |
| ---- | ----------------------------------------------------------------------------------------------------- |
| 1    | type に応じて アクティブデバイス からアクティブデバイスエントリリストを取得する。                          |

type の値が "printer" の場合、activeDevices.Printer を使用する。

| type 値            | 参照先                         |
| ------------------ | ------------------------------ |
| "printer"          | activeDevices.Printer          |
| "scanner"          | activeDevices.Scanner          |
| "cash_changer"     | activeDevices.CashChanger      |
| "customer_display" | activeDevices.CustomerDisplay  |
| "payment"          | activeDevices.Payment          |
| "drawer"           | activeDevices.Drawer           |
| "keyboard"         | activeDevices.Keyboard         |

| 項番 | 処理概要                                                                                              |
| ---- | ----------------------------------------------------------------------------------------------------- |
| 2    | リストが null または空の場合、空の DeviceSpec を返却する。                                           |
| 3    | os と大文字小文字無視で一致するエントリをフィルタリングする。                                        |
| 4    | フィルタ結果のIDリストを取得する。                                                                   |
| 5    | デバイス一覧 から type・ID・OS が一致する最初のデバイスを返却する。該当なしの場合は空の DeviceSpec を返却する。 |
