## 概要

Windows環境でNamedPipe経由でHost側のOPOS OCXを制御し、CAFIS Arch決済端末を操作するストラテジクラス。
端末接続（Open→Claim→Enable）、切断（Disable→Release→Close）、疎通確認、決済実行（JSON→ASI→DirectIO）、再印字の操作を提供する。

## メソッド詳細

### メソッド名: 端末接続

#### 戻り値

| 型              | 概要           |
| --------------- | -------------- |
| Task\<string\>  | 決済端末の応答 |

#### 引数

| 修飾子 | 型          | 引数名（論理） | 引数名（物理） | 概要                                         |
| ------ | ----------- | -------------- | -------------- | -------------------------------------------- |
| -      | DeviceSpec? | デバイス設定   | config         | デバイス仕様（nullの場合は既存Deviceを使用） |

#### 発生例外

| 型        | 概要                                       |
| --------- | ------------------------------------------ |
| Exception | 名前付きパイプ通信で例外が発生した場合     |

#### 処理概要

| 項番 | 処理概要                                                                                           |
| ---- | -------------------------------------------------------------------------------------------------- |
| 1    | 基底クラスの `端末接続(config)` を呼び出す。                                                         |
| 2    | デバイス設定からIPアドレスとポート番号で接続情報文字列を生成する。                                 |
| 3    | 接続情報が空でない場合、`CAFIS_Start;{接続情報}` コマンドを送信する。                             |
| 4    | 接続情報が空の場合、`CAFIS_Start` コマンドを送信する。                                           |
| 5    | 応答を返却する。例外発生時は `Task.FromException` で伝播する。                                   |
