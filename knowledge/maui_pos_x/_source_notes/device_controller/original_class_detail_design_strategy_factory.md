## 概要

クラス名からストラテジーインスタンスを登録・生成する汎用ファクトリークラス。
型パラメータ TBase はストラテジーのベースインターフェース（例: `IPrinterStrategy`, `ICashChangerStrategy` 等）を表す。
リフレクションを用いて `InitInstance` メソッドの有無を判定し、適切なインスタンス生成パスを選択する。

#### 発生例外

| 型  | 概要 |
| --- | ---- |
| -   | -    |

#### 処理概要

| 項番 | 処理概要                                                                                                            |
| ---- | ------------------------------------------------------------------------------------------------------------------- |
| 1    | 型Tからリフレクションで `InitInstance` メソッド（public static）を検索する。                                        |
| 2    | `InitInstance` が存在し、戻り値が型T、引数が DeviceSpec 型1つの場合:                                                |
| 2.1  | `InitInstance` をリフレクション経由で呼び出す生成関数をストラテジーマッピングに登録する。                             |
| 3    | `InitInstance` が存在しない場合:                                                                                    |
| 3.1  | `Activator.CreateInstance` でインスタンスを生成する生成関数を登録する。                                             |
| 3.2  | 生成したインスタンスが DeviceStrategyBase の場合、`InitInstance(device)` で初期化する。                             |

### メソッド名: ストラテジー生成

#### 戻り値

| 型    | 概要                       |
| ----- | -------------------------- |
| TBase | ストラテジーインスタンス   |

#### 引数

| 修飾子 | 型         | 引数名（論理） | 引数名（物理） | 概要                     |
| ------ | ---------- | -------------- | -------------- | ------------------------ |
| -      | string     | クラス名       | name           | 登録済みストラテジー名   |
| -      | DeviceSpec | デバイス仕様   | device         | デバイス仕様情報         |

#### 発生例外

| 型        | 概要                                                   |
| --------- | ------------------------------------------------------ |
| Exception | 指定されたクラス名がマッピングに未登録の場合にスローする |

#### 処理概要

| 項番 | 処理概要                                                                                       |
| ---- | ---------------------------------------------------------------------------------------------- |
| 1    | クラス名 が null または空文字列の場合、default（null） を返却する。                                |
| 2    | ストラテジーマッピングから name に対応する生成関数を取得する。                                 |
| 3    | 未登録の場合、`"Class '{name}' not registered for {TBase名}"` のメッセージで例外をスローする。  |
| 4    | 登録済みの場合、生成関数に device を渡してインスタンスを生成し返却する。                        |
