## Sheet 2. 見積概要

### 2.1 見積概要

本見積では、次世代タブレットPOS に関する共通基盤の関連実装、ドキュメント作成、テストケース整理、内容確認、レビュー反映を対象としています。2026年5月から7月までは、主要な技術内容をレビュー可能な資料として整理することを中心に進める形になります。

対象範囲は、デバイス・Host、ローカル状態保存、ログ・Sentry、テストケース、レビュー対応の単位で整理しています。各成果物の内容、工数、実施時期が確認しやすいように、3か月の範囲でまとめています。

| 項目 | 工数 | 備考 |
| --- | ---: | --- |
| 算出工数 | 6.25人月 | 3か月で月あたり1.5〜2.5名程度の作業量として算出 |
| 提示工数 | 4.00人月 | 2026年5月に2.00人月、2026年6月〜7月に2.00人月を目安として提示 |
| VTI側調整分 | 2.25人月 | 本提案範囲ではVTI側で調整する形としております |
| 2026年5月提示工数 | 2.00人月 | 初月に必要となるHost関連資料と共通基盤整理を中心に配分 |
| 2026年6月〜7月提示工数 | 2.00人月 | 仕様書、テストケース、レビュー反映、最終整理を中心に配分 |

| グループ | 対象内容 |
| --- | --- |

| レビュー・調整 | ドキュメントレビュー、Q&A、小規模な修正、提出前の成果物整理を対象とします |

### 2.5 対象外・補足事項

| 項目 | 内容 |
| --- | --- |
| 大量の新規業務画面 | 現在の対象はドキュメントと共通基盤が中心のため、本見積範囲には含めません |
| 詳細情報が未提供のDeviceまたはOCX | 仕様、環境、確認方法について個別の整理が必要になります |
| production環境での全デバイス確認 | 現地確認が必要な場合は、別途テスト計画を確認する形になります |
| 組織全体のSentry運用設計 | 本見積ではアプリケーション側で利用する範囲を対象とします |
| 確認済み範囲を超える大きな変更 | 影響範囲を確認したうえで、必要に応じて追加見積を行います |
| 工数の見直し | 成果物、対象デバイス、確認環境、レビュー回数に大きな変更がある場合は、工数とスケジュールを再確認する場合があります |
