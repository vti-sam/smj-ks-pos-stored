# Report Artifact Output Folder Convention

Report-generation skills were updated so generated artifacts go under an `output/` folder directly inside the report bundle folder.

Applies to:

- Draw.io workflow generated `.yaml`, `.drawio`, `.svg`, `.png`, `.qa.json`, and temporary previews.
- DOCX generated from Markdown by `skills/docx-builder`.
- Excel generated by `skills/program-spec-excel`, `skills/testcase-excel`, and `skills/checklist-excel`.

`output/` is ignored by both root `.gitignore` and `project-store/.gitignore`, so generated deliverables do not accumulate in Git by default.

Example local folder created during the task:

- `project-store/artifacts/reports/architecture/ARCH-03_タブレットPOS_デバイス接続サーバー構造設計書/output/`
